## âš™ï¸ How to Run
1.  Clone the repository:
    ```bash
    git clone [https://github.com/malak-hussein/MiniMart.git](https://github.com/malak-hussein/MiniMart.git)
    ```
2.  Navigate to the project directory.
3.  Configure the **Connection String** in `appsettings.json` to match your SQL Server instance.

